Jobs4Interns Application

This is a guide to run the J4I application

Requirements

  1. Install Visual Studio Code
  2. Install Git
  3. Install NodeJS (version 16.4.0 or later [check with 'node --version'], npm version 7.17.0 or later [check with 'npm -version']) --> These versions seem to work without a hitch, there are some bugs with later versions of npm.
  4. Install MongoDB and make sure it is running in the background (Administrative Tools --> Services)
  5. Install Robo3T, or any MongoDB GUI like Mongo Compass.

How to run the application

  1. Clone the repository onto your computer, and navigate to it in the terminal.
  2. Inside the jobs4intern-J4i folder, run 'npm update --force' This will update chokidar, there is a bug with node versions 14 and later with chokidar. This will also execute an npm install.
  3. Start the app in the terminal with 'npm start'. Alternitively, the app can be started by opening the debugger and clicking 'Launch Program'.
  4. To stop the program in the terminal, press CTRL + C.
